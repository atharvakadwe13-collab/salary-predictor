# 💰 Data Science Salary Predictor

A machine learning web application that predicts
Data Science salaries based on experience level,
employment type, remote ratio and company size.

## 🛠️ Built With

* Python
* Scikit-learn (Linear Regression)
* Streamlit
* Pandas
* Matplotlib & Seaborn

## 🚀 Features

* Interactive salary prediction
* Visual salary analysis by experience
* Visual salary analysis by remote ratio
* Live dataset preview
* Model accuracy display
